Guia para fazer uma horta orgânica

Neste artigo ensinaremos como fazer uma horta orgânica simples e de pequeno porte para lugares onde o espaço é limitado.

1.Reservar uma área para construir a horta, não é necessário ser um espaço grande, pode ser uma área de 1m². Você pode utilizar algum material para cercá-la, como tijolo. Porém, este local deve receber luz solar por pelo menos 4 horas.

https://images.app.goo.gl/Ty3Lf

2.Após realizar a primeira etapa, você pode adicionar terra adubada no cercado. Caso queira economizar, a terra do quintal é perfeitamente utilizável. Entretanto, ela é uma terra pobre (sem nutrientes), sendo assim, para otimizar a qualidade dos alimentos, é necessário um cuidado extra com a terra. Recomendação: pulverizar calcário.

3.Se o calcário foi pulverizado, mexa a terra para espalhar ele de forma uniforme no solo. Espere por pelo menos uma semana para começar a plantar, o solo precisa de um tempo para absorver o calcário.

https://images.app.goo.gl/tXyhBKLcDC4T9KxN8

4. Antes de plantar, utilize adubo orgânico ou, se preferir uma forma barata, folhas secas das árvores, casca de ovo, entre outros. Os nutrientes fornecidos pelo adubo melhoram a qualidade do alimento. Após realizar as etapas anteriores, plante seus vegetais.
