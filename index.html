Guia para fazer uma horta orgânica

Neste artigo ensinaremos como fazer uma horta orgânica simples e de pequeno porte para lugares onde o espaço é limitado.

1.Reservar uma área para construir a horta, não é necessário ser um espaço grande, pode ser uma área de 1m². Você pode utilizar algum material para cercá-la, como tijolo. Porém, este local deve receber luz solar por pelo menos 4 horas.
